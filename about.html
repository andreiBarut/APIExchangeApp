This website was created using Free foreign exchange, crypto rates & EU VAT Rates API The website has two functionalities:

1. The converter functionality, which is in the Home Tab. This converts an amount the user puts in from one currency to another and displays the converted amount in the page.

2. The historical dates functionality, which is in the Historical Dates tab. This functionality converts one selected currency to another and displays the value of the converted currency in a selected date range.

3. Also, there is the currency container, which can be found in the bottom of each page, displaying the exchange rate for RON in that specific day.
